## Instalasi

1. **Clone Repository**

    ```bash
    git clone https://github.com/blackjac7/algoritma-test-eigen.git
    cd algoritma-test-eigen

    ```

2. **Jalankan Program**

    Pastikan Anda sudah memiliki Node.js terinstal. Kemudian jalankan perintah berikut untuk menjalankan programnya:

    example:
    ```bash
    node jawaban1.js
    ```
